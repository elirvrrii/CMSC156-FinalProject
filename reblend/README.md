
# FlavorFuse

## 🚀 Getting Started

The following section provides a guide for users and developers alike to access the app locally.
### Prerequisites

List the software and tools you need installed before running the app.
*   [Flutter SDK](https://docs.flutter.dev/get-started/install)
*   Android Studio (for Android emulation)
*   A Firebase project configured for Flutter

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/elirvrrii/CMSC156-FinalProject
   cd reblend
   ```
   
2. Install necessary dependencies:

   ```bash
   flutter pub get
   ```

3. Set up Firebase

- Create a Firebase project
- Enable Authentication and Cloud Firestore
- Download and add the Firebase configuration files:
    - `google-services.json` for Android
    - `GoogleService-Info.plist` for iOS

3. Run the app

   ```bash
   flutter run
   ```
